# 设备图片资源说明

请从 Figma 设计稿中下载空调设备图片并保存到此目录：

## 需要的图片文件：

1. **air-conditioner.png** - 空调设备主图

## 下载步骤：

1. 在 Figma 中打开设计稿（节点 ID: 7-189 或 8-247）
2. 选择空调设备图片组件（equipment 图层）
3. 右键选择 "Export" 或使用导出功能
4. 选择 PNG 格式，建议尺寸：560x200 或更高分辨率（2x）
5. 保存到 `assets/equipment/air-conditioner.png`

## 注意事项：

- 图片应该是透明背景的 PNG 格式（如果设计稿中有透明部分）
- 建议使用 2x 或 3x 分辨率以适配不同屏幕
- 确保图片尺寸与设计稿中的比例一致
