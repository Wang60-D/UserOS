# 图标资源说明

请从 Figma 设计稿中下载以下图标并保存到此目录：

## 需要的图标文件：

1. **cool.png** - 制冷模式图标（雪花图标）
2. **heat.png** - 制热模式图标（太阳图标）
3. **dehumidify.png** - 除湿模式图标（水滴图标）
4. **fan.png** - 送风模式图标（风扇图标）

## 下载步骤：

1. 在 Figma 中打开设计稿
2. 选择对应的图标组件
3. 右键选择 "Export" 或使用导出功能
4. 选择 PNG 格式，建议尺寸：64x64 或更高分辨率
5. 保存到 `assets/icons/` 目录，使用上述文件名

## 注意事项：

- 图标应该是透明背景的 PNG 格式
- 建议使用 2x 或 3x 分辨率以适配不同屏幕
- 确保图标颜色与设计稿一致
